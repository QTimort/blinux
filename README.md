- Launch VirtualBox
- Go to File -> Import Appliance...
- Select blinux.ova
- You can now start the VM
- Login : root
  Password : root
- Then open a terminal and type blinux-setup, fill the fields
- Reboot and login to the freshly created user and you're done!
